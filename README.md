# Phishing Detection in the Age of Generative AI
### Notre Dame Data Club x Booz Allen Hamilton | Spring 2026

## Project Overview
As phishing techniques evolve, attackers are increasingly leveraging Large Language Models (LLMs) to generate convincing and scalable phishing emails. This project, conducted in partnership with **Booz Allen Hamilton**, focuses on building robust detection models to identify phishing threats and assessing how traditional detection systems handle the linguistic patterns of AI-generated content.

Our primary goal was to build a model that not only distinguishes between legitimate and phishing emails but also maintains high performance across both human-authored and LLM-generated datasets.

## Key Features
* **Multi-Class Classification:** Detection across four categories: Human Legitimate, Human Phishing, LLM Legitimate, and LLM Phishing.
* **Benchmark Analysis:** Comparison of classical Machine Learning (XGBoost) vs. Deep Learning (Transformers).
* **Robustness Testing:** Evaluation of model performance under distribution shifts (human-trained models tested on AI data).

## The Data
The dataset includes a curated mix of:
* **Human-authored emails:** Traditional legitimate and phishing communications.
* **LLM-generated emails:** Synthetic content generated by models trained on human data, designed to mimic realistic phishing attempts.

## Methodology: DBTL Cycles
The project followed a **Design-Build-Test-Learn (DBTL)** framework:
1.  **DBTL 1-2:** EDA, preprocessing, and initial feature extraction.
2.  **DBTL 3 (Classical ML):** Implementation of XGBoost with hyperparameter tuning.
3.  **DBTL 4 (Deep Learning):** Fine-tuning a **DistilBERT** transformer model (66M parameters) from Hugging Face.

## Performance Results
The fine-tuned BERT model consistently outperformed classical ML models, particularly in sensitivity and handling the nuances of LLM-generated text.

| Model | Task | Metric (AUC-PR) |
| :--- | :--- | :--- |
| **XGBoost** | Binary Classification | 0.94 |
| **Fine-Tuned DistilBERT** | Binary Classification | **0.98** |
| **Fine-Tuned DistilBERT** | Multi-Class | **0.96** |

**Key Insight:** Transformers are significantly more effective at capturing the structural and tonal differences between human and AI-generated phishing content.
